# House Price Prediction ML App

## Tech Stack
- Python
- Scikit-learn
- Flask

## Steps
1. Train model: python train.py
2. Run app: python app.py
3. Call API: /predict

## Input
{
  "size": 10
}

## Output
{
  "predicted_price": 1000
}
